## ✨ Features

* **🤖 AI-Powered Generation:** Uses Google's advanced `gemini-3.1-pro-preview` / `gemini-2.5-flash` models to craft realistic travel plans.
* **💰 Smart Budgeting:** Distributes total group budgets realistically across accommodation, food, activities, and transport.
* **🗺️ Geographic Clustering:** Organizes daily activities logically to minimize travel time between locations.
* **⚡ Blazing Fast:** Built on Vite for incredibly fast local development and optimized production builds.
* **🎨 Modern UI:** Sleek, responsive user interface built with customized components.
* **🛡️ Secure:** Environment variables and API keys are safely managed and hidden from the client side.

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Styling/UI:** CSS / Custom UI Components (Shadcn/Tailwind)
* **AI Engine:** Google Generative AI (Gemini API)
* **Hosting:** Vercel
